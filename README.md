# Maguito Sales Control Application

## Overview
Maguito is a sales control application designed to streamline sales processes for businesses. It allows users to manage sales records, track product inventory, and generate detailed reports, contributing to better decision-making.

## Features
- User-friendly interface for managing sales.
- Real-time inventory tracking.
- Ability to generate sales reports and analytics.
- Secure user authentication.
- Integration with various payment gateways.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/MercametLR/Maguito.git
   cd Maguito
   ```
2. Install the necessary dependencies:
   ```bash
   npm install
   ```
3. Set up your environment variables by creating a `.env` file based on the `.env.example` provided in the repository.
4. Start the application:
   ```bash
   npm start
   ```

## Usage
- **Login:** Users can log in using their credentials. 
- **Manage Sales:** The dashboard provides options to add, edit, and delete sales records.
- **Inventory Management:** Access the inventory section to update products and view stock levels.
- **Reports:** Utilize the reporting features to analyze sales trends and performance metrics.
